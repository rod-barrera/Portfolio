INTRODUCCIÓN

Mi trayectoria en el desarrollo web comenzó en 2008, cuando tenía alrededor de 11 años. Empecé creando páginas web relacionadas a Habbo y Counter-Strike, en las cuales editaba código HTML, CSS y JavaScript de manera básica, pero con una buena comprensión de su funcionamiento.

Durante mi adolescencia, mantuve un interés constante en la informática, aunque sin una dirección clara. En 2023, decidí realizar un curso intensivo en desarrollo web moderno, lo cual me motivó a profundizar en este campo de manera autodidacta y a estudiar la carrera de Ingeniería en Informática.

Actualmente, me dedico a crear aplicaciones web completas y funcionales, aplicando tecnologías modernas y buenas prácticas de desarrollo para asegurar proyectos robustos y eficientes.
